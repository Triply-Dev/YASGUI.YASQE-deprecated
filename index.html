About YASHE

YASQE is part of the regular YASGUIYASQE is part of the regular YASGUI web application. YASQE provides a simple syntax highlighted text area, bundled with features such as autocompletion, and the option to query SPARQL endpoints.
YASHE is a fork for the simpler ShEx syntax.
var yashe = YASHE(document.getElementById("showcase"));
(or, if you would like to instantiate YASHE from an existing text area, use var yashe = YASHE.fromTextArea(document.getElementById('textAreaItem'));
